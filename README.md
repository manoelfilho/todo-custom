# ToDo

Aplicativo de gerenciamento de tarefas customizável. Permite selecionar a priodidade de cada tarefa e estilizar de acordo com os temas disponíveis nas configurações. O app tem três cores padronizadas para os modos light e dark mode além de opções para seleção do tipo do ícone da tela inicial
